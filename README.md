# 纪念日 - DayMaster

一个在 GitHub Pages 上托管的纪念日网页，灵感来自 DayMaster App。

## 功能

- 记录重要的纪念日，自动计算已过去的天数
- 显示下一个里程碑目标及进度条
- 里程碑标记（100天、200天、365天、521天、1000天、1314天等）
- 支持多个纪念日同时管理
- 支持添加、编辑、删除纪念日
- 自定义 Emoji 图标和主题色
- 深色/浅色主题切换
- 数据存储在浏览器本地

## 部署到 GitHub Pages

1. Fork 或 clone 本仓库
2. 推送到你的 GitHub 仓库
3. 进入仓库 Settings > Pages
4. 选择 `main` 分支，根目录 `/`
5. 访问 `https://<你的用户名>.github.io/<仓库名>`

## 本地运行

直接用浏览器打开 `index.html` 即可，无需构建工具。
